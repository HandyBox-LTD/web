# handyman (web MVP)

## Requirements

- Node.js 20+
- pnpm

## Setup

```bash
pnpm install
```

## Run the web app

```bash
pnpm dev
```

Then open <http://localhost:3000>.

## Run Storybook

```bash
pnpm storybook
```

Then open <http://localhost:6006>.

## Lint / Format

```bash
pnpm lint
pnpm format
```
